# Less is nerd Shopify theme

Tema Shopify Online Store 2.0 creato per Less is nerd.

## Cosa contiene

- Homepage personalizzabile con hero minimal nerd, prodotti in evidenza, testo brand e newsletter.
- Template per prodotto, collezione, carrello, pagina generica e 404.
- Header e footer collegati ai menu Shopify.
- Stile responsive per desktop e mobile.
- Impostazioni colore modificabili dall'editor Shopify.

## Come collegarlo a Shopify

1. Apri Shopify Admin.
2. Vai in `Online Store` > `Themes`.
3. Scegli `Add theme` > `Connect from GitHub`.
4. Seleziona questa repository e il branch `main`.
5. Apri `Customize` per configurare testi, prodotti, menu e immagini.
